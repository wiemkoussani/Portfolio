# Wwiem Koussani — Portfolio

Professional portfolio for an AI software engineer. Built with React + Vite. Free hosting on Vercel.

## Run locally

```bash
npm install
npm run dev
```

## Deploy (free)

1. Create a GitHub repo and push this project.
2. Go to [vercel.com](https://vercel.com), sign in with GitHub, import the repo.
3. Deploy — you get a free URL like `your-name.vercel.app`.

## Update contact links

Edit `src/data.ts` (`email`, `linkedin`, `github`).

## Update Wini_Maint video later

In `src/data.ts`, find the `wini-maint` project, set `comingSoon: false`, and add `video: "/videos/your-file.mp4"`.
